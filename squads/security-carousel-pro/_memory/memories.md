# Squad Memory: Security Carousel Pro

## Estilo de Escrita
- Profissional, técnico, direto e baseado em fatos.
- **Idioma:** Sempre escrever em Português (Brasil).


## Design Visual
- Estilo: Modern Dark Tech.
- Cores: Deep Blue, Black, Neon Cyan.

## Estrutura de Conteúdo
- Carrosséis de 7 slides com gancho e CTA.

## Proibições Explícitas
- Não usar prompts de imagem fofos ou amigáveis; preferir estética tecnológica "séria".

## Técnico (específico do squad)
- Usa Playwright para postar no LinkedIn.
- O editor Pulse do LinkedIn (2026) apresenta instabilidade na detecção automatizada do campo de título (H1), por vezes requerendo intervenção via Browser Subagent para garantir o "commit" do texto antes da publicação.
